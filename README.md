## 产品特点
- 📦 开箱即用，将注意力集中在组件开发和文档编写上
- 📋 一体化封装的 UI 组件，让后端开发远离前端的复杂
- 📱 多终端使用支持，让使用者不必带着电脑跑
- 🏷 命令行生成基础开发模块，减少复制粘贴操作
- 🎨 文件化式菜单权限，减少数据化式的操作冗余
- 📡 内部 hook 式低耦开发，即插即用，应用即插件


## 环境需求

- PHP >= 7.4.0
- Mysql >= 5.7.0
- Composer >= 2.0.0
- Laravel >= 8.0.0
- Redis PHP Extension

## 产品安装

```shell
// 创建站点目录
$ mkdir library && cd library

// 安装项目
$ composer create-project duxphp/ravelcms

// 启动web服务访问安装向导
http://localhost
```
